# CalcHashDir
Simple Tool C# to calculate Hashes into a Directory and Comparate with another hash.

## Usage
  CalcHashDir.exe {Directorio} {Hash}
  
  CalcHashDir.exe {Directorio}
  
  CalcHashDir.exe {Path Archivo} {Hash}
  
  CalcHashDir.exe {Path Archivo}
### Examples:
  
    CalcHashDir.exe C:\System
  
  Compare Hash with file\s:
    
    CalcHashDir.exe C:\System\vccorlib120.dll BDDF9F6B23FE4B85B9EB63DF613FB777
Report Save in the directory "Reportes" in 2 type of files (CSV, JSON)
